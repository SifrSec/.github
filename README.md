### Training Blueprints & Labs

Our primary repositories serve as structured, self-paced learning pathways:

#### [Cybersecurity Homelab Builder](https://github.com/Afnaan-Ahmed/cybersecurity-homelab)
A comprehensive guide to deploying a local, virtualized Red vs. Blue simulation lab. 
* **The Learning Goal:** Master defensive engineering by attacking your own infrastructure and observing the results.
* **Core Skills Taught:** Deploying **Splunk SIEM**, capturing active telemetry, configuring log forwarders, analyzing **Wireshark** packet captures, and mapping attacks to the **MITRE ATT&CK** framework.

#### [Project Overhaul: Penetration Testing CLI](https://github.com/Afnaan-Ahmed/Project-Overhaul)
An educational CLI framework designed to lower the barrier of entry for syntax-heavy offensive utilities. It abstracts complex parameters into guided workflows to teach users how tools string together during an engagement.
* **Tool Integrations Included:** `Nmap`, `Amass`, `Gobuster`, `Metasploit`, `Sqlmap`, `Hashcat`, and `John the Ripper`.

#### [Hardware Security & HID Attack Lab](https://github.com/Afnaan-Ahmed/Raspberry-Pi-Pico-HID-Attack)
A physical security training resource demonstrating keystroke injection vulnerabilities using the low-cost Raspberry Pi Pico.
* **The Learning Goal:** Understand BadUSB mechanics, learn to write custom payload scripts via MicroPython, and explore endpoint defense strategies against unauthorized USB human interface devices.

#### [CompTIA Security+ Mastery Vault](https://github.com/Afnaan-Ahmed/CompTIA-Security-plus-Resource-Vault)
A highly curated archive of study resources, mapping documentation, and essential bookmarks customized directly for the SY0-701 exam blueprint. Ideal for students establishing their fundamental security baseline.
